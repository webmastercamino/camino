Recommended color adjustments

Fix: Darken amber for text uses → #8B5510

Swap --amber: #BA7517 to #8B5510 for inline text (eyebrows, labels, links). Keep the original #BA7517 only for button backgrounds and decorative blocks. The eyebrow text "Faith · Community · Hope" is the most visible failure — it's small and amber on white.

Fix: Darken amber button → #9B6213 for small button text

If any button uses font-size under 18px (or under 14px bold), the white-on-amber combo fails AA. Darken to #9B6213 for buttons — still reads as amber/gold, but crosses the 4.5:1 threshold.

Keep: Teal #0F6E56 — excellent contrast on all backgrounds

Passes AA at 5.4:1 on white. Works for text, buttons, icons. No change needed.

Keep: Coral #993C1D — strong contrast, great brand warmth

6.4:1 on white. Passes all WCAG levels including AAA for large text. Fits the Honduran earth-tone palette perfectly.

Consider: Add a deeper amber for dark backgrounds / banners

For full-bleed amber/teal sections, #FAC775 (--amber-mid) on teal gives a warm pop. Good for CTAs in the footer CTA bar. Contrast = 3.8:1 on teal — fine for large bold text.

Mobile/browser: All colors are CSS custom property safe

The system-ui font stack + CSS variables approach works in Safari iOS 15+, Chrome Android, Samsung Browser, and all modern browsers. No font CDN calls = no render-blocking. Warm gray (#f5f4f0) renders identically across all major browsers. No fixes needed here.
